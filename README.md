# Cornedeon Mono

Inspired by [Le Chiffre Keyboard](https://github.com/tominabox1/Le-Chiffre-Keyboard)

Maintainer: [A.L.Ko](mailto:alko@alk0.ru) -> [github](https://github.com/alko-kbd/cornedeon-mono)

Web Site: [cornedeon.ru](https://cornedeon.ru)

Hardware: Handwired, RP2040


## Build

0. Prepare Vial build environment.

1. cd vial-qmk/keyboards/alko

2. git clone https://github.com/alko-kbd/cornedeon_mono

3. qmk compile -kb alko/cornedeon_mono -km vial
